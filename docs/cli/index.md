## <a name="installation"></a>Installation

Install it using [npm](https://www.npmjs.com/package/@codiac.io/codiac-cli):

```bash
$ npm install -g @codiac.io/codiac-cli
```

...then you check your install using: 
```bash
$ codiac version
```
...and 
```bash
$ codiac help
```
